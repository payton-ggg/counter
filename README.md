    <h2>Overview</h2>
    <p>The Counter project is a simple application built with React and Vite. It serves as a basic example of how to create a counter that increments and decrements a value with user interaction.</p>

    <h2>Features</h2>
    <ul>
        <li>Increment and decrement buttons</li>
        <li>Display the current count</li>
        <li>Responsive design</li>
    </ul>

    <h2>Technologies Used</h2>
    <p>This project utilizes the following technologies:</p>
    <ul>
        <li><strong>React</strong> - A JavaScript library for building user interfaces</li>
        <li><strong>Vite</strong> - A fast build tool and development server</li>
        <li><strong>CSS</strong> - For styling the application</li>
    </ul>

    <h2>Getting Started</h2>
    <p>To get started with this project, follow the instructions below:</p>
    
    <h3>Prerequisites</h3>
    <p>Make sure you have the following installed:</p>
    <ul>
        <li>Node.js (v12 or later)</li>
        <li>npm or yarn</li>
    </ul>

    <h3>Installation</h3>
    <pre><code>git clone <repository-url>
cd counter-project
npm install</code></pre>

    <h3>Running the Project</h3>
    <pre><code>npm run dev</code></pre>
    <p>Open your browser and go to <strong>http://localhost:3000</strong> to see the app in action.</p>

    <h2>Usage</h2>
    <p>Click the "Increment" button to increase the count and the "Decrement" button to decrease it. The current count will be displayed in the center of the application.</p>
